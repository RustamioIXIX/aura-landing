## Что нужно сделать тебе

### 1. Установить зависимости

Открой терминал в папке проекта и выполни:

```powershell
npm.cmd install
```

Если PowerShell ругается на политики выполнения, используй именно `npm.cmd`, а не `npm`.

### 2. Запустить локально

```powershell
npm.cmd run dev
```

После этого открой адрес, который покажет Vite. Обычно это:

```txt
http://localhost:5173
```

### 3. Проверить адаптив

Проверь сайт в DevTools на ширинах:

- `1440px`
- `1024px`
- `768px`
- `430px`
- `390px`

### 4. Собрать production-версию

```powershell
npm.cmd run build
```

Готовая версия появится в папке:

```txt
dist/
```

### 5. Залить на хостинг

Проще всего:

1. создать репозиторий на GitHub;
2. загрузить проект;
3. подключить его в Vercel;
4. нажать Deploy.
